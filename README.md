# Shared Kitchen v3

Modern rebuild of the Shared Kitchen app — Next.js (React) + Prisma + MySQL, deployed on Railway.

## Status
- [x] Phase 1: Minimal Next.js starter (verify deploy pipeline)
- [ ] Phase 2: Connect MySQL, migrate data
- [ ] Phase 3: Build features (auth, money, menus, shop, cuisine, AI)

## Deploy (Railway)
1. Push this repo to GitHub.
2. Railway -> New Project -> Deploy from GitHub repo -> pick this repo.
3. Railway auto-detects Next.js, builds, and serves on its assigned PORT.
4. Add a MySQL database in the same project for Phase 2.
